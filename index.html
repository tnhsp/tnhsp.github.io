6.PHP程序员现在就业薪资怎么样？

在北京PHP开发工程师的平均薪水已经到14200元（数据统计来职友集），而且目前还保持强劲的发展势头。

7.PHP只能开发网站，是不是很单一呢？就不能开发出别的么?

PHP技术是专门做网站开发的，请不要小看网站开发，它包括的范围是很广的: 比如论坛(天涯，清华水木等)、电子商务网(淘宝、ebay等)、sns网站(比如世纪佳缘、百合网、人人网等)、cms（内容管理系统）网站(比如优酷、土豆、中关村下载网站)， 世界上排名100的网站都使用到PHP技术，你说单一吗? 而且每个网站中还有很多应用（比如支付，短信，彩信等，webservice）等也要使用到PHP技术？你说单一吗?

PHP技术是专门做WEB网站开发的，所有只要是做上面说的网站开发，PHP技术就是公司的优先考虑，目前一些网站都开始选用PHP来改写. LAMP(linux /Unix+ apache + mysql 数据库 + PHP) 或者 LNMP(linux /Unix+ Nginx + mysql 数据库 + PHP)。
